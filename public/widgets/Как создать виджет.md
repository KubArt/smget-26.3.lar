
### 3. Справка по разработке новых виджетов

> **Как создать новый виджет:**
> 1. **Название класса:** Должно быть в формате `window.SmWidget_тип_виджета` (где тире заменены на подчеркивания).
> 2. **Наследование:** Обязательно `extends SmWidget`.
> 3. **Метод `mount()`:** Это точка входа. Базовый класс сам вызовет его после проверки всех задержек и скролл-триггеров.
> 4. **Использование `this.track('событие')`:** Для аналитики.
> 5. **Пример структуры:**
> ```javascript
> window.SmWidget_my_type = class extends SmWidget {
>    mount() {
>        this.injectStyles();
>        this.createContainer(this.assets.html);
>        this.container.querySelector('.btn').onclick = () => this.track('click');
>    }
> }
> ```
