Мы организуем файлы следующим образом:

    app/Metrics/Contracts/MetricDriverInterface.php — общий стандарт.

    app/Metrics/Drivers/YandexMetrikaDriver.php — реализация для Яндекса.

    app/Metrics/Drivers/VkPixelDriver.php — реализация для ВК.

    app/Metrics/MetricsManager.php — "мозг", который распределяет события.

    app/Metrics/GoalsSynchronizer.php — сервис для автоматического создания целей в сторонних API.
